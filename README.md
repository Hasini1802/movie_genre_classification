# Movie Genre Classification

## Project Overview

This project is developed as part of the CodSoft Machine Learning Internship.

The objective of this project is to build a Machine Learning model that predicts the genre of a movie based on its plot summary.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Naive Bayes Classifier

## Dataset

IMDB Genre Classification Dataset

Dataset Link:
https://www.kaggle.com/datasets/hijest/genre-classification-dataset-imdb

## Machine Learning Techniques

* Text Preprocessing
* TF-IDF (Term Frequency–Inverse Document Frequency)
* Multinomial Naive Bayes

## Project Workflow

1. Load Dataset
2. Clean and Prepare Data
3. Convert Text into Numerical Features using TF-IDF
4. Train Naive Bayes Classifier
5. Predict Movie Genres
6. Evaluate Model Performance

## Output

The model predicts the genre of a movie using its plot summary.

Example:

Input:
"A young boy discovers magical powers and attends a wizard school."

Predicted Genre:
Fantasy

## Conclusion

This project demonstrates how Natural Language Processing (NLP) and Machine Learning can be used to classify movie genres based on textual descriptions.
